This is the Readme for the application Music Player. 

Instructions for use:

Unzip the folders included in MusicPlayer.zip to the main Music Player Directory.

You should have directories that look like this:

"C:\Music Player\Frog"
"C:\Music Player\Long"
"C:\Music Player\Loop"
"C:\Music Player\Short"
"C:\Music Player\Med"
"C:\Music Player\Songs"

From there, run the file. 

Have fun!
